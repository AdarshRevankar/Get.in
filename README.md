# Get.In

Get.In is an Android app for browsing and booking event slots. Users can view a list of available events, see event details like location and date, browse available time slots for each event, and book a slot by providing their name and phone number. After booking, users receive a confirmation screen with a success message.

## Features 
1. Material You Design
2. Clean Interface
3. Day and Night Mode
4. Modern MVVM Architecture
5. Dependency Injection with Hilt
